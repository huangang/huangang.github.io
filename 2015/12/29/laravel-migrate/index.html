laravel migrate 实践

2015年12月29日

运行migrate命令

php artisan migrate

创建一个叫article的表

php artisan make:migration create_articles_tables --create=articles

编辑定义article需要的字段

再次运行migrate命令,进行创建

php artisan migrate

修改article字段

php artisan make:migration add_intro_column_to_articles --table=articles

再次运行migrate命令,进行修改

php artisan migrate
如果需要运行$table->dropColumn('intro');还需要导入一个包运行composer require doctrine/dbal

migrate回滚命令

php artisan migrate:rollback

laravel tinker

php artisan tinker
